# Budget Tracker App

A simple, beautiful Android app to track daily expenses, set a monthly budget, and view spending analytics.

## Features
- Dashboard with monthly budget, total expenses, and remaining balance
- Add, view, and filter expenses (by category/date)
- Set or edit monthly budget
- Pie chart analytics for spending by category
- Local data storage with Room (no network required)
- Clean MVVM architecture, Hilt DI, Paging 3, and XML layouts

## How to Run
1. **Clone the repository**
2. **Open in Android Studio** (Arctic Fox or newer recommended)
3. **Build the project** (all dependencies are included)
4. **Run on an emulator or device** (minSdk 21+)

## Notes
- All strings are extracted to `strings.xml` for easy localization
- No login or authentication required
- No network calls or unnecessary libraries
